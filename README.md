# AP7200_MDK
Исходники ядра с возможностью компиляции lk.bin, preloader, модулей и т.п.

Полная сборка
------------
./mk new

Сборка ядра
------------
./mk new kernel

сборка bootimage
-------------
./mk bootimage

Сборка модулей
------------
./mk r k path_to_module

Сборка lite kernel
------------
./mk n lk
